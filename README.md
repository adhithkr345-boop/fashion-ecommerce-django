# Fashion E-Commerce Website

A full-stack Fashion E-Commerce web application built using Django and Python. This project allows users to browse products, manage carts, and place orders through an interactive web interface.

## Features

- User Registration & Login
- Product Catalog Management
- Shopping Cart Functionality
- Order Processing
- Admin Dashboard
- Product Image Management
- Responsive User Interface

## Tech Stack

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQLite
- Git & GitHub

## Installation

1. Clone the repository

git clone https://github.com/adhithkr345-boop/fashion-ecommerce-django.git
2. Navigate to the project directory

cd fashion-ecommerce-django
3. Install dependencies

pip install -r requirements.txt
4. Run the development server

python manage.py runserver
5. Open your browser and visit:

http://127.0.0.1:8000/
